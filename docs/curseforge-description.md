# Rolling Pin Awards

Rolling Pin Awards gives a guild a lightweight way to hand out Burnt and Golden Rolling Pins in game.

Use it to nominate guildmates, let the guild react, and leave the final call to the ranks you trust with moderation. Officers can approve nominations, issue awards directly, manage rank permissions, and clean up award history when needed.

The addon keeps the fun visible without turning it into a spreadsheet. It includes:

- Burnt and Golden rolling pin award types
- guild nominations with advisory voting
- officer moderation tools
- direct awards for authorized ranks
- award history and leaderboard views
- character mapping for alts and nicknames
- guild-scoped sync between addon users
- a draggable minimap button and custom in-game UI

Rolling Pin Awards is built for guild communities that want a little ceremony, a little accountability, and a reason to laugh after the pull goes sideways.
